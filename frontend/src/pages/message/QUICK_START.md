# 消息中心快速开始

## 访问消息中心

### 方式一：通过顶部导航栏
1. 登录系统后，在顶部导航栏右侧可以看到消息图标（铃铛图标）
2. 如果有未读消息，图标上会显示红点和未读数量
3. 点击消息图标即可进入消息中心

### 方式二：直接访问
在浏览器地址栏输入：`http://localhost:5173/messages`

## 主要功能使用

### 1. 查看消息列表
- 消息列表按时间倒序排列
- 未读消息会以红色标签和加粗字体显示
- 已读消息显示为灰色标签

### 2. 筛选消息
**按消息类型筛选**:
- 点击"消息类型"下拉框
- 选择：审批通知、提醒通知、系统通知或业务通知

**按阅读状态筛选**:
- 点击"阅读状态"下拉框
- 选择：未读或已读

**按时间范围筛选**:
- 点击日期范围选择器
- 选择开始日期和结束日期

**关键词搜索**:
- 在搜索框输入关键词
- 按回车或点击"搜索"按钮

### 3. 标记消息为已读

**单个消息标记**:
1. 找到要标记的未读消息
2. 点击该消息行的"标记已读"按钮
3. 消息状态会立即更新为已读

**批量标记**:
1. 勾选要标记的消息（可多选）
2. 点击顶部的"批量标记已读"按钮
3. 选中的消息会全部标记为已读

**全部标记**:
1. 点击"全部标记已读"按钮
2. 在确认对话框中点击"确定"
3. 所有未读消息会被标记为已读

### 4. 消息跳转
1. 点击消息标题
2. 如果消息关联了业务页面，会自动跳转
3. 未读消息会在跳转前自动标记为已读

### 5. 删除消息

**单个消息删除**:
1. 找到要删除的消息
2. 点击该消息行的"删除"按钮
3. 在确认对话框中点击"确定"

**批量删除**:
1. 勾选要删除的消息（可多选）
2. 点击顶部的"批量删除"按钮
3. 在确认对话框中点击"确定"

### 6. 刷新消息
点击"刷新"按钮可以手动刷新消息列表和未读数量

## 消息类型说明

| 类型 | 标签颜色 | 说明 |
|------|---------|------|
| 审批通知 | 蓝色 | 审批相关的通知，如待审批、审批通过等 |
| 提醒通知 | 橙色 | 各类提醒，如合同到期、里程碑提醒等 |
| 系统通知 | 绿色 | 系统级通知，如系统维护、版本更新等 |
| 业务通知 | 紫色 | 业务相关通知，如门店开业、工程完工等 |

## 实时更新

系统会每30秒自动检查新消息，并更新：
- 顶部导航栏的未读消息数量
- 消息中心页面的未读消息统计

## 快捷键（计划中）

未来版本将支持以下快捷键：
- `Ctrl + M`: 打开消息中心
- `Ctrl + R`: 刷新消息列表
- `Ctrl + A`: 全选消息
- `Delete`: 删除选中的消息

## 移动端使用

消息中心支持移动端访问：
1. 在移动设备上登录系统
2. 点击顶部的消息图标
3. 所有功能在移动端都可正常使用
4. 界面会自动适配移动设备屏幕

## 常见问题

**Q: 为什么看不到消息图标？**
A: 请确保已经登录系统。消息图标只在登录后显示。

**Q: 未读消息数量不更新怎么办？**
A: 系统每30秒自动更新一次。您也可以点击"刷新"按钮手动更新。

**Q: 删除的消息可以恢复吗？**
A: 不可以。删除操作是永久性的，请谨慎操作。

**Q: 如何关闭某类消息的通知？**
A: 当前版本暂不支持消息通知设置，该功能将在后续版本中提供。

**Q: 消息可以导出吗？**
A: 当前版本暂不支持消息导出，该功能将在后续版本中提供。
