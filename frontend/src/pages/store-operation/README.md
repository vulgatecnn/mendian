# 门店运营管理模块

## 页面说明

### PaymentTracking.tsx - 付款追踪页面
- 门店付款记录管理
- 付款状态追踪
- 付款提醒功能
- 付款统计分析

### AssetManagement.tsx - 资产管理页面  
- 门店资产档案管理
- 资产状态追踪
- 资产维护记录
- 资产报废处理

## 功能特性

- 支持多维度查询和筛选
- 支持数据导入导出
- 支持移动端适配
- 集成审批流程
- 实时数据更新

## 权限控制

- store_operation.payment.view - 查看付款信息
- store_operation.payment.edit - 编辑付款信息
- store_operation.asset.view - 查看资产信息
- store_operation.asset.edit - 编辑资产信息
