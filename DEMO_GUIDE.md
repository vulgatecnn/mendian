## 🚀 快速启动指南

### 环境要求
- Node.js 18+
- npm 或 pnpm
- 现代浏览器 (Chrome, Firefox, Safari, Edge)

### 启动步骤

1. **安装依赖**
   ```bash
   cd frontend
   npm install
   # 或使用 pnpm
   pnpm install
   ```

2. **启动开发服务器**
   ```bash
   npm run dev
   # 或使用 pnpm
   pnpm dev
   ```

3. **访问地址**
   - 前端地址: `http://localhost:5173`
   - 默认用户: `admin / admin123`
